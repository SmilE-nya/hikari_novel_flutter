## 功能

- Material Design 3风格
- 支持深浅模式切换
- 适配平板
- 支持章节缓存
- 支持查看评论与回复
- 阅读进度保存
- 网格列数自定义（1-6列）
- 浏览历史/用户书架列表/网格布局切换
- 书架列表模式显示章节简介
- 已缓存小说导出为 .txt / .epub（支持自定义导出目录）
